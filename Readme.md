# Frontend: <https://sheroes-front.herokuapp.com/>

```
- cd frontend/
- npm install
- npm run start
```

# Backend: <https://sheroes-form.herokuapp.com/>

```
- cd backend/
- pipenv shell
- pipenv install
- python manage.py makemigrations
- python manage.py migrate
- python manage.py runserver
```
